. ประเภทของ Skill ที่สำคัญ
stitch-design: เป็นจุดเริ่มต้น (Entry point) สำหรับงานดีไซน์ ช่วยขยายความ Prompt ให้มีความเป็น UI/UX มากขึ้น และสร้าง Design System

design-md: ทำหน้าที่วิเคราะห์โปรเจกต์แล้วสร้างไฟล์ DESIGN.md เพื่อสรุปกฎเกณฑ์การออกแบบ (Design Tokens, Typography, Colors) ให้กลายเป็น "Source of Truth" ของโปรเจกต์

react-components: ทำหน้าที่แปลง (Convert) หน้าจอที่ออกแบบไว้ใน Stitch ให้กลายเป็นโค้ด React หรือ TypeScript ที่พร้อมใช้งาน โดยยังคงความถูกต้องตาม Design Tokens

stitch-loop: เป็น Skill ระดับสูงที่ช่วยสร้าง "Multi-page website" ทั้งเว็บจากการสั่งงานเพียงครั้งเดียว (Single Prompt) โดยมันจะวนลูปสร้างหน้า วางโครงสร้างไฟล์ และตรวจสอบความถูกต้องให้อัตโนมัติ

enhance-prompt: ช่วยเกลาคำสั่งที่คลุมเครือให้กลายเป็นคำสั่งที่ Optimized สำหรับการสร้าง UI โดยเฉพาะ

remotion: ใช้สำหรับสร้างวิดีโอ Walkthrough หรือ Demo ของหน้าจอแอปที่ออกแบบไว้
