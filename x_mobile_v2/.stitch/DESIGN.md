# XInvest V2 Design System: Financial Atelier

## Core Philosophy
"The Digital Private Bank" - A high-end editorial investment experience. Sharp, mathematical typography meets soft, layered surfaces. The interface should feel like a premium financial journal.

## Design Tokens

### Colors
- **Primary**: `#004532` (Deep Emerald - The bedrock of the brand)
- **Surface**: `#FFFFFF` (High-grade paper white)
- **Background**: `#F8F9FA` (Subtle tonal contrast)
- **Text Primary**: `#111827` (Matte Black)
- **Text Secondary**: `#6B7280` (Steely Gray)
- **Accent**: `#10B981` (Vibrant Emerald for positive growth)
- **Border**: `#E5E7EB` (Whisper Gray)

### Typography
- **Display/Headers**: `Manrope` (SemiBold/Bold) - For a modern, geometric feel.
- **Body/Utility**: `Inter` (Regular/Medium) - For maximum legibility.
- **Micro-copy**: `Inter` (Medium, Tracking: +0.02em).

### Shape & Layout
- **Radii**: `16px` for cards/containers, `8px` for buttons.
- **Spacing**: 8pt grid system. Standard page padding: `24px`.
- **"No-Line" Rule**: Avoid heavy borders. Use tonal layering and shadows to define depth.
- **Shadows**: Soft, multi-layered shadows (`0 4px 6px -1px rgb(0 0 0 / 0.1)`).

### Components
- **Editorial Cards**: Large typography, generous whitespace, asymmetrical details.
- **Micro-Animations**: Subtle fades and 20px Y-axis slides on entry.
- **Tonal Dividers**: Subtle background shifts rather than lines.

## Guidelines
1. **Premium Whitespace**: Never crowd the content.
2. **Asymmetrical Balance**: Use white space as a structural element.
3. **Data as Art**: Charts (fl_chart) should be clean, line-based, and monochromatic emerald.
